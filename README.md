# reddelstiltskin
Firefox extension that switches 'www.reddit.com' to 'old.reddit.com'. Simple and privacy-centered, will never be sold.

## Currently supports
- redirecting 'www.reddit.com' to 'old.reddit.com'
- changing '/posts' to '/submitted'
